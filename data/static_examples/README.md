# Static Example Bundles

This folder contains fixed sample CSV and PDF assets for every industry tab shown in the workshop customization page.

Structure:

- `telecommunications/`
- `retail/`
- `manufacturing/`
- `insurance/`
- `finance/`
- `energy/`
- `education/`
- `hospitality/`
- `logistics/`

Each scenario folder contains:

- `tables/` for CSV files
- `documents/` for PDF files
- `config/sample_questions.txt` for a fixed question set aligned to the scenario

These files are intentionally static so the documentation can reference real repository assets with filenames that exactly match the examples shown in `workshop/docs/02-customize/index.md`.
