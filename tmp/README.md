# tmp/

這個目錄現在只保留仍有長期參考價值的整理文件，用於交接、治理、翻譯維護與少量未來研究整理。

## 保留原則

- 保留仍可作為交接依據的治理決策
- 保留可重複使用的翻譯與寫作標準
- 保留已完成交付的整體摘要與少量後續事項
- 若需保留 future work 研究，應集中在單一研究總檔，而不是拆成多份中間規劃稿
- 不再保留逐項 work item、一次性盤點報告、審稿錯誤清單或分散的中間規劃稿

## 目前實際保留文件

- `project-governance-zhtw.md`：專案治理、內容來源、教學架構與能力邊界
- `translation-standards-zhtw.md`：翻譯範圍、術語、風格規則與 reviewer checklist
- `project-governance-zhtw.md` 末段 `Current State / Last Consolidation`：本輪交付的完成項目、主要產出、驗證結果與少量後續事項
- `future-work-research.md`：集中整理 future work、Agent365 與 hosted agent 等延伸研究內容

## 不屬於這個目錄的內容

- 正式使用者文件：請看 `workshop/docs/`
- MkDocs 建置輸出：請看 `workshop/site/` 或 root `site/`
- 一次性審稿與盤點材料：完成後應刪除，不回留在 `tmp/`

若未來再開新的工作流，請優先更新上述整理文件，而不是重新累積大量 task-by-task 的暫存筆記。
