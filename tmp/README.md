# tmp/

這個目錄現在只保留仍有長期參考價值的整理文件，不再保留逐項 work item、一次性盤點報告或已完成任務的中間紀錄。

保留原則：

- 保留仍可作為交接依據的治理決策
- 保留可重複使用的翻譯與審稿標準
- 保留已完成交付的整體結論、驗證結果與後續少量待辦
- 刪除已完成的 task notes、gap analysis、smoke test 原始記錄與中間規劃稿

目前保留文件：

- `project-governance-zhtw.md`：專案治理、內容來源、教學架構與必要/選配能力邊界
- `translation-standards-zhtw.md`：翻譯範圍、術語、風格規則與 reviewer checklist
- `implementation-summary-zhtw.md`：本輪交付的完成項目、主要產出、驗證結果與少量後續事項

正式使用者文件來源仍在 `workshop/docs/`。此目錄不會作為 MkDocs 使用者站台的內容來源。

若未來再開新的工作流，請優先更新上述整理文件，而不是重新累積大量 task-by-task 的暫存筆記。
